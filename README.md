# 🧭 Indoor Navigation System (Offline + AI Assisted)

Ever got lost inside a hospital, mall, or college building where Google Maps just gives up? Yeah… same 😅

This project is an indoor navigation system that works **without GPS** by using your phone’s sensors — kind of like how fitness apps count your steps, but smarter.

It tracks your movement, estimates your position, and helps you navigate inside buildings. Oh, and there's a bit of AI involved too 😉

---

## 🚀 What Makes It Cool?

- 👣 **Step-Based Navigation**
  - Uses your phone like a fitness tracker
  - Counts steps + direction to estimate where you are

- 📍 **No GPS Needed**
  - Works perfectly indoors where GPS fails

- 🧠 **AI-Powered Assistance**
  - Integrated with OpenRouter (free API)
  - Helps with smarter navigation and interactions

- 🗺️ **Offline Maps**
  - No internet? No problem.
  - Preloaded maps keep things running

- 🔄 **Live Movement Tracking**
  - Your position updates as you walk

## ⚙️ How It Works (Simple Version)

1. Pick where you want to go
2. Set your starting point
3. Walk 🚶
4. System tracks:
   - Steps
   - Direction
   - Distance
5. Boom — your position updates in real time
